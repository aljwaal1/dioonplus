# دفتر الديون المتقدم

تطبيق Flutter عربي لإدارة الديون والتسديدات بدون إنترنت، بتصميم احترافي متكامل.

## ما الجديد في هذا التحديث

- **هوية بصرية احترافية**: تدرّج كحلي/تركواز غامق مع لمسة ذهبية للأزرار والعناصر المهمة، بدل الألوان المسطحة السابقة.
- **هيدر متدرّج (Hero Header)**: يعرض الصافي الإجمالي وبطاقتين زجاجيتين (Glass chips) لـ "لي عند الناس" و"علي للناس" بشكل مباشر وواضح.
- **بطاقات مرتفعة بظل ناعم**: بدل الإطارات المسطحة، مع شريط لون جانبي يدل على نوع العملية.
- **أزرار احترافية**: زر رئيسي بتدرّج ذهبي مع تأثير ضغط وظل، وزر عائم (FAB) متدرّج، وشريط تنقل سفلي عائم بشكل كبسولة (Pill) بدل الشريط التقليدي.
- **اختيار نوع العملية بشكل تفاعلي**: شبكة أزرار ملوّنة بالأيقونات بدل القائمة المنسدلة.
- **حركات دخول ناعمة**: كل عنصر في القوائم يظهر بحركة "ظهور وارتفاع" خفيفة.
- **مؤثرات صوتية احترافية وخفيفة**: صوت تأكيد عند الحفظ، صوت ناعم عند التنقل والنقر، وصوت عند الحذف — مع زر صغير في الهيدر لكتم الصوت، وكل الأصوات محفوظة محليًا (لا تحتاج إنترنت).
- **أيقونة تطبيق وشاشة بدء (Splash) مخصصتان**: شعار محفظة بتدرّج كحلي وذهبي، يتولّد تلقائيًا أثناء البناء عبر GitHub Actions.
- **تنسيق أرقام أوضح**: فواصل الآلاف في عرض المبالغ (مثل 1,250 د.أ).

## الميزات الأساسية

- الشخص يظهر مرة واحدة في الرئيسية.
- ملخص: لي عند الناس، علي للناس، الصافي.
- إضافة دين له أو دين علي.
- إضافة تسديد منه أو تسديد مني.
- صفحة تفاصيل لكل شخص تعرض كشف حساب وتسلسل زمني.
- نسخ رسالة تذكير جاهزة لإرسالها في واتساب أو الرسائل.
- حفظ البيانات محليًا على الجهاز.

## طريقة البناء عبر GitHub

ارفع الملفات إلى GitHub، ثم شغل workflow باسم:

`Build APK Direct`

أثناء البناء يتم تلقائيًا:

1. تركيب الحزم (`flutter pub get`).
2. توليد أيقونة التطبيق من `assets/icon/icon.png`.
3. توليد شاشة البدء (Splash) من `assets/icon/splash_logo.png`.
4. بناء ملف APK بصيغة release.

بعد نجاح البناء سيظهر ملف APK مباشرة داخل:

`apk/debt-advanced.apk`

وسيظهر ملف:

`DOWNLOAD_APK.md`

وفيه رابط التحميل المباشر.

## ملاحظات

- التطبيق Android فقط.
- لا يحتاج إنترنت أثناء الاستخدام، بما في ذلك الأصوات والأيقونة (كل شيء محفوظ محليًا داخل التطبيق).
- لا يحتاج صلاحيات حساسة.
- الحد الأدنى المقترح: Android 6 وما فوق.
